# Text Preview WinForms

Компонент на C# Windows Forms для предварительного просмотра текста с настраиваемым шрифтом и цветом.

## Функциональность

- Ввод текста в текстовое поле
- Выбор шрифта через FontDialog
- Выбор цвета через ColorDialog
- Отрисовка текста на форме с выбранными параметрами

## Интерфейс

- textBox1 — поле ввода текста
- button1 — выбор шрифта
- button2 — выбор цвета
- button3 — отобразить текст

## Куда можно встроить

- В текстовые редакторы с предпросмотром
- В дизайнерские приложения
- В программы для создания открыток
- В инструменты для вёрстки

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Рисование происходит непосредственно на форме. При изменении размеров окна текст исчезает (для постоянного отображения использовать PictureBox или OnPaint).
